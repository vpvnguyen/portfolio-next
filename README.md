# My Portfolio built on Next.js

# Setup
1. Create a new Next.js project
> `npm init -y`
> `npm install react react-dom next`
2. Edit package.json for Next scripts
```
"scripts": {
  "dev": "next",
  "build": "next build",
  "start": "next start"
}
```
3. Create pages dir for Next to route and render pages
> `mkdir pages`
